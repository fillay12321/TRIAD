# Changelog

## [1.1.0] - 2024-06-XX
### Added
- Полная асинхронная симуляция сети TRIAD с поддержкой параллельных сценариев
- Метрики DAG, консенсуса, латентности, интерференции, батчей, атак, перегрузок
- Ограничение числа параллельных задач через семафор
- Реализация сценариев SimpleTx, ContractTx, CrossShardTx, FaultyTx, BatchTx
- Поддержка реального времени и отладочного вывода

### Changed
- Перевод всех мьютексов на tokio::sync::Mutex
- Использование StdRng для Send-совместимости
- Рефакторинг main.rs: сценарии вынесены в отдельные async-функции

### Fixed
- Исправлены deadlock-и и зависания при запуске симуляции
- Корректное ограничение числа транзакций по TARGET_TX
