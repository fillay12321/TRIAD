# Security & Attack Resistance Roadmap for TRIAD

## 1. Phase/Amplitude Attack Protection
- [ ] Ввести лимиты на amplitude для waves
- [ ] Реализовать phase randomization и phase anomaly detection
- [ ] Добавить stake-based caps на влияние волны

## 2. Nothing-at-Stake & Griefing
- [ ] Ввести penalization за destructive interference
- [ ] Реализовать reputation system для узлов
- [ ] Добавить stake slashing за атаки

## 3. MEV & Long-Range Attacks
- [ ] Реализовать probabilistic ordering для защиты от MEV
- [ ] Ввести decay старых states для защиты от long-range атак

## 4. Quantum-Resistant Cryptography
- [ ] Интегрировать lattice-based или hash-based подписи
- [ ] Провести бенчмарки производительности
